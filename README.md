# DSA-Project
Group Members : Saad Sarwar(DS012), Muhammad Hassan(DS042), Muhammad Ahmad Raza(DS011).
Project Title: House Allocation System.
Languages used : python, java, html css.
DSA implementations like Queue, Priority Queue, Linked list, Hash map etc.
Website includes Home page, Apply page(from where users submit application for houses), Waiting list page(Shows info like houses allocated, not allocated, pending), About Us page, contact page, and then Admin Login (from where admin logins and Handles addition or removal of houses and Allocation and Deallocation of houses based on priority).
MY SQL Lite Database used.
